# Mithyalabs CLI
A command-line-interface(cli) for project management.



## Create new project
1. New **web** project
	```js
	mithya new web
	```
2. New **admin** project
	```js
	mithya new admin
	```

## Generate project files
### Web Projects
1. Base models for front-end projects
	```js
	mithya gen models --out-dir='src/base'
	```
